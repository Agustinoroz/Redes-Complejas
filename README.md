# Redes-Complejas
Repositorio con implementaciones y análisis desarrollados en el marco de la materia Redes Complejas (Universidad de Buenos Aires, 2024).

El curso se centra en el estudio de sistemas complejos desde la perspectiva de redes, con énfasis en modelado, análisis estructural y aplicaciones a redes biológicas.

## Temas abordados

Análisis estructural de redes complejas

Modelado de redes aleatorias y de crecimiento

Medidas de centralidad y análisis espectral

Estructura comunitaria y detección de comunidades

Aplicaciones a redes biológicas y genómicas

## Organización del repositorio

El repositorio está organizado por temas.
Cada carpeta contiene implementaciones y análisis específicos, junto con una breve descripción del problema abordado.

## Lenguajes y herramientas

Python
Librerías de análisis de redes y computación científica
